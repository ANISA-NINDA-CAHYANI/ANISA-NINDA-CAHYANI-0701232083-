Mockup Web dengan Word

Ini adalah artikel lengkap tentang cara belajar Mockup web dengan Word untuk membuat halaman web sederhana.

HTML digunakan untuk membuat struktur halaman, sedangkan CSS digunakan untuk mempercantik tampilan.

Dalam latihan ini, kamu akan belajar membuat layout web, mengatur warna, font, serta menambahkan gambar. Dengan menguasai HTML dan CSS, kamu bisa membangun tampilan website yang menarik dan profesional.

Pelajari juga framework CSS seperti Bootstrap atau Tailwind untuk mempercepat proses desain web.
